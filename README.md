# Student Productivity Data Analysis

## Objective
To analyze how study hours affect productivity score.

## Tools Used
- Python
- Pandas
- Matplotlib

## Steps Performed
- Data Cleaning
- Missing Value Check
- Correlation Analysis
- Data Visualization

## Graphs Created
- Scatter Plot (Study Hours vs Productivity)
- Gender Distribution (Bar Chart)
- Study Hours Distribution (Pie Chart)
- Average Productivity by Study Hours (Bar Graph)

## Conclusion
Productivity score increases as study hours increase.
